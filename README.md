# Boss Constructor Match-3

Локальный конструктор боссов для игр 3-в-ряд.

Ссылка на конструктор: [Boss Constructor](https://kraupins.github.io/boss_constructor_2/)

## Что умеет

- Ввод названия босса, HP, типа босса и сложности.
- Автоматический расчёт параметров босса.
- Ручная тонкая настройка с видимыми auto-значениями.
- Расчёт урона мечей, шаров, брони, атаки и регена.
- Автосимуляция боя на поле 7×7.
- Ручной «Трай пользователя»: можно сыграть матч-3 против текущего босса прямо в конструкторе.
- Повторный запуск симуляции с теми же настройками, но новым seed.
- Рекомендации по балансу, если босс не проходится или слишком лёгкий.
- Вкладка «Итог» с копированием текстового отчёта и JSON-конфига.

### Трай пользователя

Ручной бой против текущего босса. Нажми `Начать трай`, затем кликай по тайлу и соседнему тайлу. Если swap создаёт матч 3+, ход засчитывается, применяются все формулы: урон мечей, глиняный шар, алмазный щит, бедрок-снижение брони, атака и реген босса.

Есть кнопки:

- `Подсказать ход` — показывает один хороший доступный swap;
- `Скрыть подсказку` — убирает подсветку;
- `Перемешать` — пересоздаёт поле без траты хода;
- `Новый трай` — начинает бой заново с теми же настройками.


## Как запустить

Нужен Node.js. Лучше Node.js 20+.

```bash
npm install
npm run dev
```

После запуска открой адрес, который покажет терминал. Обычно:

```text
http://127.0.0.1:5173/
```

## Как пользоваться

1. Открой вкладку **Конструктор**.
2. Введи название босса, HP, тип и сложность.
3. При необходимости измени игрока, веса тайлов и параметры симуляции.
4. Нажми **Запустить симуляцию**.
5. Открой вкладку **Симуляция** и смотри:
   - win rate;
   - первую успешную попытку;
   - среднюю длину боя;
   - лог боя;
   - советы по балансу.
6. Нажимай **Перезапустить**, чтобы прогонять новые попытки с теми же настройками.

## Как работает Auto

В полях тонкой настройки босса пустое значение означает автоматический расчёт.

Теперь рядом с каждым таким полем отображается реальное auto-значение:

- если поле пустое — параметр считается автоматически;
- кнопка `Зафикс.` записывает текущее auto-значение в поле как ручное;
- если значение введено вручную — кнопка `Auto` очищает поле и возвращает автоматический расчёт.
