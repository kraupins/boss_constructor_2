# Boss Constructor Match-3

Локальный конструктор боссов для 3-в-ряд боя MergeCraft.

## Что умеет

- Ввод названия босса, HP, типа босса и сложности.
- Автоматический расчёт:
  - HP босса с учётом сложности;
  - брони босса;
  - атаки босса;
  - интервала атаки;
  - регена;
  - урона деревянного, железного, золотого и алмазного меча;
  - урона глиняного шара;
  - защиты от алмазного шара;
  - снижения брони от бедрок шара.
- Отдельные вкладки:
  - **Конструктор** — все входные настройки;
  - **Статы** — итоговые параметры, разбитые по разделам;
  - **Симуляция** — автобой, лог и советы по балансу;
  - **Формулы** — основные формулы расчёта.
- Автоматический матч-3 автобой:
  - поле 7×7;
  - поиск доступных swap-ов;
  - выбор лучшего хода;
  - каскады;
  - проверки брони через шанс блока;
  - атака и реген босса;
  - повторный запуск симуляции с теми же настройками.
- Помощник баланса:
  - если симуляция не проходит, предлагает точечные правки;
  - если босс слишком лёгкий, предлагает усилить его;
  - если много таймаутов, предлагает изменить лимит ходов, HP или реген.

## Как запустить

Нужен Node.js. Лучше Node.js 20+.

```bash
npm install
npm run dev
```

После запуска открой адрес, который покажет терминал. Обычно:

```text
http://127.0.0.1:5173/
```

## Как пользоваться

1. Открой вкладку **Конструктор**.
2. Введи название босса, HP, тип и сложность.
3. При необходимости измени игрока, веса тайлов и параметры симуляции.
4. Нажми **Запустить симуляцию**.
5. Открой вкладку **Симуляция** и смотри:
   - win rate;
   - первую успешную попытку;
   - среднюю длину боя;
   - лог боя;
   - советы по балансу.
6. Нажимай **Перезапустить**, чтобы прогонять новые попытки с теми же настройками.

## Важная логика повторного запуска

Раньше одинаковый seed мог давать одинаковый результат. Теперь каждый повторный запуск добавляет внутренний номер прогона к seed, поэтому симуляция реально делает новый прогон, но настройки босса и игрока остаются теми же.
