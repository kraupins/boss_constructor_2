# Balance Rules

## Броня

Броня не режет урон напрямую в бою. Она работает как шанс заблокировать отдельный удар.

Пример:

```text
5 алмазных мечей, 10 урона каждый, броня босса 30%.
Для каждого меча отдельно кидается проверка.
Если 2 меча заблокировались, то проходят только 3 меча.
Итоговый урон = 3 × 10 = 30.
```

Для баланса средний урон можно считать так:

```text
ExpectedDamage = RawDamage × (1 - Armor)
```

## Мечи

```text
WoodDamage = BaseSwordDamage × 1.0
IronDamage = BaseSwordDamage × 1.7
GoldDamage = BaseSwordDamage × 2.5
DiamondDamage = BaseSwordDamage × 3.4
```

Базовый урон меча считается от HP босса, брони и целевой длины боя:

```text
BaseSwordDamage = BossMaxHP / (TargetTurns × 4.5 × (1 - BossArmor))
```

## Глиняный шар

Глиняный шар наносит чистый урон и игнорирует броню босса.

```text
ClayDamage = max(1, round(WoodSwordDamage × 0.25))
```

## Алмазный шар

Даёт временную защиту игроку до следующей атаки босса.

```text
ShieldPerBall = 0.5% × (4 / AttackInterval) × DifficultyShieldMultiplier
```

## Бедрок шар

Снижает броню босса до следующего урона мечами.

```text
BedrockBreakPerBall = clamp(BossArmor × 0.025, 0.4%, 1.25%) × DifficultyBreakMultiplier
```

## Атака босса

Атака босса делится на несколько частей. Каждая часть отдельно проверяется бронёй игрока.

```text
ChunkDamage = BossAttackDamage / AttackChunks
```

## Реген босса

```text
HealAmount = BossMaxHP × RegenPercent
```

Реген не может поднять HP выше максимального HP босса.

| Броня босса | -5% | 0 | +5% |
| Урон босса | ×0.8 | ×1.0 | ×1.2 |
| Интервал атаки | +1 ход | 0 | -1 ход |
| Реген | ×0.75 | ×1.0 | ×1.25 |
| Интервал регена | +1 ход | 0 | -1 ход |
| Урон игрока | ×1.15 | ×1.0 | ×0.85 |
| Алмазный шар | ×1.15 | ×1.0 | ×0.9 |
| Бедрок шар | ×1.15 | ×1.0 | ×0.9 |

## Генератор боссов

Генератор не выбирает случайные числа из одного общего диапазона. Он сначала выбирает этап босса:

- ранний;
- средний;
- поздний;
- финальный;
- или случайный этап.

После этого он подбирает диапазоны HP босса, HP игрока, брони, бонусов, целевой длины боя и лимита ходов. Тип босса влияет на дополнительные параметры:

- бронированный чаще получает больше броню и больший вес бедрок-шара;
- агрессивный чаще получает более полезные алмазные шары;
- регенерирующий получает больше веса сильных мечей;
- финальный получает более плотный микс сильных мечей, защиты и антиброни.

Сгенерированные значения записываются в обычные поля конструктора, поэтому после генерации их можно вручную менять, фиксировать или снова возвращать в Auto.
